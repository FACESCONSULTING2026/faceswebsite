# Videos Directory

This directory should contain the hero video for the homepage:

**hero-video.mp4** - Main hero video file (MP4 format recommended for best browser compatibility)
**hero-video.webm** - Optional WebM format for better compression (fallback will use MP4)

## Video Requirements:
- Format: MP4 (H.264 codec) or WebM
- Recommended resolution: 1920x1080 or higher
- Aspect ratio: 16:9 or wider
- File size: Keep under 10MB for faster loading (consider compression)
- Duration: Can be any length (will loop automatically)
- Content: Should work well with the blue overlay applied

The video will automatically:
- Loop continuously
- Play muted (required for autoplay)
- Cover the full hero section area
- Have a dark blue overlay applied for text readability
