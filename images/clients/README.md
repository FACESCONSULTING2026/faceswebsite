# Client Logos Directory

This directory contains client logo images for the Our Clients page.

## Required Logo Files

All logos should be placed in this directory (`/images/clients/`) with the following filenames:

1. **ploughman-analytics.jpg** - Ploughman Analytics, Inc.
2. **westbon.jpg** - Westbon
3. **umiuni.png** - UmiUni
4. **mhm-properties.png** - MHM Properties
5. **cartgogogo.png** - Cartgogogo

## Logo Specifications

- **Format**: PNG preferred, JPG acceptable (SVG acceptable if available)
- **Background**: Transparent background preferred
- **Minimum Size**: 400px wide for best quality
- **Naming**: All lowercase, kebab-case (as shown above)

## Display Sizes

- **Logo Wall**: Logos displayed at 44-56px height
- **Featured Clients**: Logos displayed at larger size (varies by layout)
- **Client Cards**: Logos displayed at 60-72px height

All logos use `object-fit: contain` to maintain aspect ratio and prevent distortion.
